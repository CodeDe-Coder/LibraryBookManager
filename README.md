# Library Book Manager

A Simple Java Console Application (Class XI Computer Science Project)
___
# Overview
The Library Book Manager is a menu-driven Java program that helps manage books in a small library.
It allows the user to:
Add new books
Search for a book
Issue a book
Return a book
Display all books
The program is written using Object-Oriented Programming (OOP) and uses ArrayList to store book records.
___

# Project Structure
LibraryBookManager/
├── Book.java
├── Library.java
└── Main.java

## Book.java
Contains the Book class with attributes:

•Title

•Author

•Issue status

## Library.java
Manages:

•Adding books

•Searching books

•Issuing & returning

•Displaying all books

## Main.java

Handles the menu and user input using Scanner.
